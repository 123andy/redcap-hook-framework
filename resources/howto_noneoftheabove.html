How to use the noneoftheabove hook

Purpose
This hook allows you to prevent multiple selections of checkbox values when one is marked as a 'none of the above' option.
Prerequisites
None
Using the hook
  • Create a field with the required choices, including an option equivalent to 'none of the above'.
  • In the Action Tags / Field Annotation
    Add the @NONEOFTHEABOVE={value} annotation (no quotes around {value})
    With {value} being the code of the 'none of the above' choice.
  • Example
    Checkbox choices
    1, Apples
2, Bananas
3, Cherries
98, None of the Above
    Action Tags / Field Annotation
    @NONEOFTHEABOVE=98
