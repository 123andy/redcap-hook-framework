How to use the shazam hook

Purpose
This hook allows you to rearrange the input elements on a page, for example in a HTML table of a descriptive field.
Prerequisites
  • A basic understanding of HTML is required to rearrange fields in an HTML table.
  • A basic understanding of Bootstrap is required to prettify the HTML table.
Using the hook
  • To add a table to rearrange fields on a form, add a descriptive field to the form.
  • In the Action Tag / Field Annotation, add @SHAZAM.
  • Add the HTML code for the table in the field label of the descriptive field. Please make sure to remove any newlines from the HTML code; they are replaced by br elements and ruin your syntax. Shazam features:
    • To move the value of a field into a span 
      <span class="shazam">field_name</span>
      Where field_name is the variable name of the field.

    • To move the label of a field (not the actual value) into a span 
      <span class="shazam">field_name:label</span>
      Where field_name is the variable name of the field.

    • To mirror the visibility of another field on the form 
      <span shazam-mirror-visibility=field_name>This will appear or hide with field_name</span>
      Where field_name is the variable name of the field.

  • The table can be prettified using Bootstrap.

  • Example
    On a form with variables first_name (variable label: First name) & last_name (variable label: Last name), the HTML code below (newlines have been added for readibility) 
    <table shazam-mirror-visibility='last_name'>
 <tr>
  <td><span class='shazam'>last_name:label</span></td>
  <td><span class='shazam'>last_name</span></td>
  <td><span class='shazam'>first_name:label</span></td>
  <td><span class='shazam'>first_name</span></td>
 </tr>
</table>
    Will result in a table similar to the one below
    Last name First name
    Which will only be visible if the last_name field would normally be visible.
    The table can subsequently be prettified using Bootstrap, e.g. by adding the class table-condensed to the table element.
    Last name First name
